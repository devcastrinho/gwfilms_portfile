# GW Films Portfolio

Site estatico pronto para publicar no GitHub Pages.

Arquivos principais:

- `index.html`
- `style.css`
- `script.js`

## Como publicar no GitHub Pages

1. Crie um repositorio no GitHub.
2. Envie `index.html`, `style.css`, `script.js` e este `README.md`.
3. No GitHub, abra `Settings > Pages`.
4. Em `Build and deployment`, escolha `Deploy from a branch`.
5. Selecione a branch `main` e a pasta `/root`.

## Video

O arquivo `paola-leandro.mp4` tem mais de 400 MB. O GitHub bloqueia arquivos acima de 100 MB, por isso ele ficou fora do upload pelo `.gitignore`.

Para manter o video no site, use uma destas opcoes:

- comprimir o video para menos de 100 MB;
- hospedar o video em outro lugar e trocar o `src` no `index.html`;
- enviar uma versao curta/leve somente para o destaque do portfolio.
